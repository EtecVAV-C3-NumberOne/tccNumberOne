Introdução à Estatística

Aprenda os conceitos fundamentais da Estatística e suas devidas implicações.

📕 Material didático
🎯 Iniciante

1. 📌 O que é Estatística?

A Estatística é a área da Matemática que coleta, organiza, analisa e interpreta dados para compreender fenômenos e ajudar na tomada de decisões. Ela é usada em praticamente tudo: pesquisas de opinião, estudos científicos, controle de produção, economia, esportes e muito mais.

Por meio de métodos estatísticos, é possível resumir grandes volumes de informações em gráficos, tabelas e medidas como média, mediana e moda, facilitando a visualização e compreensão dos dados.

Além disso, a Estatística permite fazer previsões, identificar padrões e verificar a confiabilidade de resultados, sendo essencial em contextos que envolvem incertezas ou variabilidade.

2. 🔍 Etapas do Estudo Estatístico

  • Coleta de dados: obter informações através de pesquisas ou fontes alternativas;
  • Organização dos dados: colocar os dados em tabelas, listas, gráficos para facilitar a leitura;
  • Análise dos dados: calcular medidas estatísticas e identificar padrões;
  • Interpretação e conclusão: apuração e demonstração dos resultados finais obtidos.
💡 Observação Importante

A veracidade não está intrínseca nas estatísticas, ou seja, elas podem ser verdadeiras ou não. Visando informação e conhecimentos válidos, é necessário checar a validade das informações utilizadas na estatística.

3. 📑 Tipos de Dados

Os dados podem ser classificados de duas formas:

  • Quantitativos
    • Discretos: valores inteiros contáveis (ex.: número de filhos).
    • Contínuos: valores que podem ser fracionários (ex.: altura, peso).
  • Qualitativos
    • Nominais: sem ordem lógica (ex.: cor dos olhos, sexo).
    • Ordinais: têm ordem (ex.: classificação em 1º, 2º, 3º lugar).

1. 📏 Medidas Estatísticas Fundamentais

Amplamente usados nas estatísticas, os seguintes conceitos são de suma importância para interpretação e aplicação plena dos dados coletados, possibilitando conclusões lógicas e realistas. Exemplos:

  • Média Aritmética
    • Soma de todos os valores dividida pelo número de elementos
    • x + y + z+ ... / w
      • w = quantidade de valores
      • x,y,z = valores (exemplificação)

  • Mediana
    • O valor que divide o conjunto ao meio quando ele está em ordem crescente.
      • Se o número de valores for ímpar, a mediana é o valor que se encontra entre duas metades iguais do conjunto, considerando a menor divisão possível.
      • Se o número de valores for par, a mediana é a média dos dois valores encontrados entre duas metades iguais do conjunto, considerando a menor divisão possível e diferente de zero.

  • Moda
    • O valor que mais se repete dentro do conjunto.
      • Pode não existir moda, quando todos os valores aparecem a mesma quantidade de vezes.
      • Pode haver mais de uma moda, quando encontram-se dois ou mais picos iguais nos dados — sem cobrí-los inteiramente.

2. 📋 Tabelas Estatísticas

Tabelas são um elemento comumente usado na exposição de dados, dada a praticidade de seus elementos visuais e como os organiza. Dentre a composição das tabelas, estão:

  • Título: Indica o assunto da tabela.
  • Cabeçalho: Nomes das colunas.
  • Corpo: Onde os dados estão dispostos.
  • Fonte: Origem dos dados.

3. 📊 Tipos de Gráficos

(Explicação detalhada sobre gráficos)

  • Gráficos de colunas/barras
    • Barras verticais ou horizontais
    • Usado para comparar valores
  • Gráfico de linhas
    • Pontos ligados por linhas.
    • Usado para mostrar evolução ao longo do tempo.
  • Gráfico de setores
    • Representa proporções em um círculo.
    • Cada “fatia” representa uma parte do total.
  • Histograma
    • Gráfico de barras coladas.
    • Usado para dados contínuos agrupados em intervalos.
  • Pictograma
    • Usa figuras ou símbolos para representar quantidades.

4. 🧠 Interpretação e aplicações na prática

Ao analisarmos e interpretarmos estatísticas, é essencial se assegurar do entendimento correto das informações, em alinhamento com o propósito do autor. Paralalelamente, deve-se prestar atenção às medidas usadas, valores apresentados, além de identificar tendências/padrões. Os hábitos supracitados evitam uma vasta gama de possíveis problemáticas como conclusões/resultados falaciosos, advindos de amostras mal escolhidas ou impertinentes, manipulação de dados e, finalmente, apresentação de tendências confusas e irrelevantes que suscitam desinformação.

Conceitos Estatísticos

📕 Material didático
🎯 Intermediário

1. 📏 Introdução

O Universo Estatístico (ou população) é o conceito utilizado para definir o conjunto de elementos a serem estudados ou observados em uma pesquisa/experimento. Exemplos:

  • Todos os carros de uma cidade. (Relevante para estudar acidentes de trânsito)
  • Todos as notas dos estudantes de uma escola. (Relevante para levantar médias e desempenho geral dos alunos)
  • Todos os clientes de um banco. (Relevante para estudar hábitos de consumo)
  • Todas as árvores de um parque. (Relevante para estudar doenças ou crescimento)

É essencial o Universo Estatístico ser homogêneo, ou seja, possuir elementos minimamente relacionados entre si. Tendo em mente este conceito, pode-se listar vários tópicos derivativos que compõem a estrutura das estatísticas e seu respectivo levantamento e organização de dados.

  • Variáveis
    • Nas estatísticas, as variáveis representam os objetos de estudo, os elementos do Universo Estatístico os quais a pesquisa almeja estudar.
    • Exemplo:
      • Diante do Universo Estatístico "Carros", pode-se considerar "Marca do carro" uma variável.
  • Dados Estatísticos
    • Informações coletadas acerca das variáveis presentes no Universo Estatístico, ou seja, características dos elementos presentes no Universo Estatístico com relevância para estudo/observação.
    • Exemplos:
      • Altura, idade, salário, gênero, notas.
      • Diante de um Universo Estatístico como os alunos de uma escola, podemos extrair a idade como variável e, assim, os dados estatísticos seriam as idades dos alunos. Exemplos: 14, 15, 16, 17, ...
  • Amostra
    • Subconjunto representativo de um Universo Estatístico reunido com objetivos analíticos. As amostras estatísticas são essenciais quando não é possível analisar o Universo Estatístico como um todo, possibilitando a análise da população por meio da indução de uma parcela.
    • Exemplo:
      • Diante do Universo Estatístico "Carros em circulação no Brasil", uma amostra seria 500 carros escolhidos aleatoriamente em diferentes cidades.
  • Rol
    • Organização dos dados coletados em uma amostra em ordem crescente ou decrescente.
    • Caso não sejam dados numéricos (1,2,3...), o rol será organizado baseado em outra ordem pré-determinada.
    • Exemplo:
      • Amostra numérica: 1, 2, 3... | ...3, 2, 1
      • Amostra não-numérica: Recruta, Soldado, Cabo (ordem pré-determinada do Exército Brasileiro)
